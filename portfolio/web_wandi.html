GitHub

I upload all of my projects, including this website, on my GitHub Page. You can access my GitHub Page over the portfolio or over the at the top embedded card. The website is still a work in progress.

The purpose and idea

The underlining idea for this website was to try and improve my HTML and CSS skills. I also wanted to try and create a website I could use as a portfolio and reference for future work opportunities.

I wanted to create a personal website with humor and personality. I hoped to learn new things and create something I can be proud of.

HTML and CSS

This site contains 3 main HTML-Sites and some more sub HTML-Sites. I don't really use complex HTML. One of my HTML-Sites contains a Head-Attribute for clarification, a NAV-Attribute for Navigation and a Main-Part of the Site for the content of the site.

The CSS on this site contains of the main style.css Stylesheet and the asset stylesheet from Bootstrap. First all HTML Attributes are styled, and then my classes come next. After the classes the responsive part of this site is coded. First, I struggled a lot with the responsive parts of this site, but after getting some help it all starts to look better now.

Hosting, Assets and Embed

The site gets hosted by 000webhost. I choose this site because they offer to host one website for free. The domain gets hosted by hostpoint.ch. I choose hostpoint.ch because it's a simple and well-made hosting services for cheap Swiss domain.

I used Bootstrap as an Asset for the site. The portfolio page was the Album Example. I changed the CSS to suit better to my site.

I embedded some things. I used the YouTube Share Feature to get the Embed-Code for various videos. I also used GitHub-Cards from Hsiaoming Yang.
